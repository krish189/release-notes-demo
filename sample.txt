Feature Branch Demo
-------------------
This file demonstrates adding content to a feature branch for testing purposes.

Steps:
1. Create a new branch.
2. Add this text file.
3. Commit and push changes.
4. Raise a pull request to main.
5. Merge the PR and check the release notes artifact.

Expected Outcome:
- A release-notes.md file will be generated by the workflow.
- The artifact will include PR details and commit messages.
